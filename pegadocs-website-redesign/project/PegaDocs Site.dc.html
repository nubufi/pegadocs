HOW IT WORKS

Four connections. One assistant.

PegaDocs is the wiring between the tools you already run. Plug each layer in — it stays yours, in your cloud.

01

Connect models

Point at any LLM & embedding provider — OpenRouter, Bedrock, Azure OpenAI. Your keys, hot-swappable.

02

Attach vector store

Bring Pinecone, pgvector, Qdrant or Weaviate. PegaDocs handles chunking, indexing & retrieval.

03

Sync data sources

Connect SharePoint, S3 buckets, Confluence or Postgres. Scheduled syncs keep answers fresh.

04

Chat anywhere

Talk to it from a web widget, Microsoft Teams, Telegram or Slack — grounded, cited answers.

INTEGRATIONS

Bring your own everything

No lock-in. Swap any layer without touching the rest. Don't see yours? It's a small adapter away.

SELF-HOST

Up and running in five minutes

One container, one config file. Runs on your laptop, a VM, or Kubernetes — your data never leaves your network.

✓100% open source — Apache-2.0 licensed
✓No telemetry, no phone-home, no vendor account
✓Single binary & Docker image, declarative YAML config
bash — pegadocs 
# clone & launch
git clone https://github.com/nubufi/pegadocs
cd pegadocs

# configure your providers in pegadocs.yaml
docker compose up -d

✓ models     azure-openai, bedrock
✓ vectors    pinecone://prod-kb
✓ sources    sharepoint, s3://hr-docs
✓ channels   web, teams, telegram

→ PegaDocs ready at http://localhost:8080
